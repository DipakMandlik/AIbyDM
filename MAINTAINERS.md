## Maintainer Duties

- Triage new issues and pull requests.
- Keep labels, milestones, and project boards useful.
- Review changes in the assigned area.
- Protect contributor experience by enforcing the Code of Conduct.
- Keep dependencies and GitHub Actions healthy.

## Availability

Maintainers are volunteers. If a maintainer is unavailable for an extended
period, another maintainer can temporarily cover their review area.
