## [0.1.0] - 2026-06-16

### Added

- First public open-source release for AIByDM.
- Premium repository README with hero, badges, screenshots, demo preview, roadmap, and onboarding links.
- Top-level community, roadmap, and architecture guides.
- Documentation hubs for getting started, development, deployment, contributing, design system, and content system.
- Demo asset structure under `assets/demo/` and screenshot gallery under `assets/screenshots/`.
- Release workflow for versioned GitHub releases.

### Changed

- Aligned repository metadata and documentation around the platform vision: Learn AI. Build AI. Master AI.
- Tightened contribution guidance for content, code, docs, and community workflows.
- Improved release documentation and GitHub setup guidance for a contributor-ready public repository.

### Infrastructure

- Kept validation, deployment, CodeQL, and dependency review workflows in the public release surface.
- Prepared GitHub Pages, release notes, discussions, and repository discoverability for open-source growth.
