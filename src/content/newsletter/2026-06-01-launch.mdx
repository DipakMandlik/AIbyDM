## Welcome to AIByDM

AIByDM is live with a practical path for learning artificial intelligence from the ground up. The first content set focuses on foundations: Python for AI work, essential mathematics, local model experimentation, orchestration frameworks, vector databases, and quick checks that help learners measure progress.

## What Is Included

- Foundations lessons for Python basics and math for AI.
- Tool guides for Ollama, LangChain, and Qdrant.
- A Python basics exam to validate core concepts.
- Interactive practice entries for quizzes and interview flashcards.

## Next Up

The next content pass will expand the foundations track with machine learning basics, model evaluation, prompt engineering, and beginner RAG workflows. The goal is to keep every topic practical, testable, and connected to tools learners can actually run.
