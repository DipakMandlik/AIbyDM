# Support

AIByDM is a community-driven open source learning platform. The fastest way to
get help depends on what you need.

## Where to Ask

| Need              | Use                                                                                    |
| ----------------- | -------------------------------------------------------------------------------------- |
| General questions | [GitHub Discussions](https://github.com/DipakMandlik/AIbyDM/discussions)               |
| Bug reports       | [Bug report issue form](https://github.com/DipakMandlik/AIbyDM/issues/new/choose)      |
| Feature ideas     | [Feature request issue form](https://github.com/DipakMandlik/AIbyDM/issues/new/choose) |
| Content requests  | [Content request issue form](https://github.com/DipakMandlik/AIbyDM/issues/new/choose) |
| Security concerns | Follow [SECURITY.md](SECURITY.md)                                                      |

## Response Expectations

- Maintainers aim to triage new issues within 3 business days.
- Pull requests should receive an initial review or maintainer note within 5
  business days.
- Volunteer availability may vary. Clear reproduction steps and focused PRs help
  reviews move faster.

## Before Opening an Issue

1. Search existing issues and discussions.
2. Confirm the problem still exists on the latest `main` branch when possible.
3. Include links, screenshots, logs, or examples that help maintainers reproduce
   the issue.
