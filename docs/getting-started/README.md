# Getting Started

## Prerequisites

- Node.js `22.12.0+`
- npm `10+`

## Local Setup

```bash
git clone https://github.com/DipakMandlik/AIByDM.git
cd AIByDM
npm install
npm run dev
```

Open `http://localhost:3000/AIByDM/` after the dev server starts.

## First Contribution Checklist

1. Pick one product area: Learn, Tools, Games, Exams, Newsletter, Community, or Search.
2. Check existing issues or discussions before larger changes.
3. Keep content slugs explicit and stable.
4. Run `npm run typecheck`, `npm run lint`, and `npm run build` before opening a PR.
5. Include screenshots for visible UI changes.
