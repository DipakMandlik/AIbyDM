# Getting Started

## Prerequisites

- Node.js `22+`
- pnpm `10.33.0+`

## Local Setup

```bash
git clone https://github.com/DipakMandlik/AIbyDM.git
cd AIbyDM
pnpm install
pnpm dev
```

Open `http://localhost:4321` after the dev server starts.

## First Contribution Checklist

1. Confirm the area you want to improve.
2. Check whether there is already an issue or discussion.
3. Run `pnpm validate` before opening a PR.
4. Include screenshots for UI or README changes.
